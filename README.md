# 学習タスク管理アプリ

学校の予習・復習・課題・テスト勉強を一元管理するWebアプリケーションです。

## 概要

このアプリは、学生が学校の学習タスクを効率的に管理するためのツールです。以下の機能を提供します：

- 📚 **科目管理**: 科目情報の登録・管理
- ✅ **タスク管理**: 予習・復習・課題・テスト勉強の管理
- 📊 **進捗追跡**: タスクの進捗状況と完了率の可視化
- 📝 **学習ログ**: 学習時間と内容の記録
- 📅 **テスト管理**: テスト日程と範囲の管理
- 📈 **分析機能**: 学習時間や進捗のグラフ分析
- 💾 **データバックアップ**: データのエクスポート・インポート機能

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **グラフ**: Recharts
- **ストレージ**: LocalStorage (ブラウザ)

## セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/naokiwork/idea-23-assignment-management-app.git

# プロジェクトディレクトリに移動
cd idea-23-assignment-management-app

# 依存関係をインストール
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリを確認できます。

### ビルド

```bash
# プロダクションビルド
npm run build

# プロダクションサーバーの起動
npm start
```

## 主な機能

### 1. 科目管理
- 科目の追加・編集・削除
- 担当教員、年度・学期の管理

### 2. タスク管理
- 予習・復習・課題・テスト勉強の登録
- サブタスクによる詳細な進捗管理
- 期限切れタスクの警告表示

### 3. 学習ログ
- 学習時間の記録
- 日付別・科目別の集計
- 学習内容の記録

### 4. テスト管理
- テスト日程と範囲の管理
- テストまでの残り日数の表示
- テスト関連タスクの自動生成

### 5. 分析機能
- 学習時間のグラフ表示
- 科目別の学習時間分析
- タスク完了率の可視化
- 月次進捗の追跡

### 6. データ管理
- データのエクスポート（JSON形式）
- データのインポート
- バックアップ・復元機能

## データ構造

アプリは以下のデータ構造を使用します：

- **Subjects**: 科目情報
- **Tasks**: タスク情報
- **Subtasks**: サブタスク情報
- **StudyLogs**: 学習ログ
- **Exams**: テスト情報
- **Timetables**: 時間割情報
- **Classes**: 授業回情報

詳細は `study_app_spec_v0.2.md` を参照してください。
